# HSK Master

A comprehensive web application for learning Chinese through the HSK (Hanyu Shuiping Kaoshi) examination system, covering levels 1 through 6.

## Overview

HSK Master is an interactive learning platform designed to help students master Chinese vocabulary and prepare for HSK exams. The application provides structured learning paths, vocabulary lists, practice quizzes, and progress tracking for all six HSK levels.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No installation or build process required!

### Running the Application

1. Clone the repository:
```bash
git clone https://github.com/hoangchien97/HSK-Master.git
cd HSK-Master
```

2. Open `index.html` in your web browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply double-click the `index.html` file in your file explorer.

### Using the Application

1. **Select a Level**: Click on any HSK level (1-6) from the Levels section
2. **Study Vocabulary**: Review the complete vocabulary list with Chinese characters, pinyin, and English translations
3. **Take Quizzes**: Test your knowledge with interactive multiple-choice quizzes
4. **Track Progress**: View your learning statistics and progress in the Progress section

## HSK Level Information

### HSK 1 (Beginner)
- 150 words
- Basic greetings and simple phrases
- Foundation for Chinese communication

### HSK 2 (Elementary)
- 300 words (cumulative)
- Simple daily conversations
- Basic social interactions

### HSK 3 (Intermediate)
- 600 words (cumulative)
- Discuss daily life, work, and studies
- More complex sentence structures

### HSK 4 (Upper Intermediate)
- 1,200 words (cumulative)
- Fluent discussions on various topics
- Read simple Chinese texts

- Read Chinese newspapers and magazines
- Watch Chinese television and films
