# Sprint 5 — Grading + Student Dashboard (Progress)

## Goal
Teacher chấm bài, trả feedback; student dashboard hiển thị progress thật (attendance + assignments) + widgets.

## Scope
### 1) Grading (Teacher/Admin)
- Submissions inbox: pending/graded
- Grade: score + feedback
- Return to student
- Allow resubmit rules

### 2) Student dashboard
- Roadmap HSK1–6 (lock/unlock rule stub)
- Next lesson card (stub)
- Upcoming schedule list (real)
- Achievements/leaderboard (stub data)
- Stats: attendance rate, assignment completion

## Acceptance Criteria
- [ ] Teacher grade & student sees score/feedback
- [ ] Dashboard shows real computed stats
